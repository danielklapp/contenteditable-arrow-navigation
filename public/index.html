contenteditable-arrow-navigation

A utility to make multiple content editable elements easier to navigate with arrows.
  • Github Repo
  • NPM Package

Click on any contenteditable element and use the up and down arrows to navigate.

This can be used with or without the CSS property of `word-wrap: pre-wrap;` Which shows whitespace and line breaks.

This is an early development, there are likely bugs to work out. If you come across any please create a github issue. Pull request are welcome.

To see list of known bugs see github issues.
